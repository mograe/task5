Свои API ключи вводите в api_keys.py

Команды:
/start - подписка
/categorys - управление категориями
/keywords - управление ключевых слов
/news - вывод новостей

REST API

/user
POST(аргументы - user-id(айди пользователя телеграмм)) - добавление в список пользователей указанный айди

/subscriptions/categories/
GET(аргументы - user-id(айди пользователя телеграмм)) - вывод категории пользователя
POST(аргументы - user-id(айди пользователя телеграмм), categories(желаемая категория, принимает только одну из списка ниже)) - подписка пользователя на категорию
DELETE(аргументы - user-id(айди пользователя телеграмм)) - отписка пользователя на категорию

Список категорий:
Категория : аргумент,что бы задать необходимую категорию
'Бизнес':'business', 
'Развлечение':'entertainment', 
'Здоровье':'health', 
'Наука':'science', 
'Спорт':'sports', 
'Технологии':'technology'

/subscriptions/keywords/
GET(аргументы - user-id(айди пользователя телеграмм)) - вывод ключевого слова пользователя
POST(аргументы - user-id(айди пользователя телеграмм), keywords(ключевое слово)) - подписка пользователя на ключевое слово
DELETE(аргументы - user-id(айди пользователя телеграмм)) - отписка пользователя на ключевое слово

/news
GET(аргументы - user-id(айди пользователя телеграмм)) - вывод новостей
